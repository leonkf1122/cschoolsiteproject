C Basics

A basic C program has a structure with three parts: The header files, the Main function and the return statement. Theh header files are standard libraries that provide functions like Input/Output. Now, the Main function is where everything actually happens in the program and everything is executed with satements. A statement is an istruction given to the computer ending in a semicolon at the end of each one. Lastly, in the main function, there is a return statement. return 0; in main() indicates a sucessful execution.
